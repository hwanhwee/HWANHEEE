# 조환희 디지털 포트폴리오

이 프로젝트는 Streamlit으로 제작된 디지털 포트폴리오입니다.

## 주요 기능
- 프로젝트 갤러리 및 상세 설명
- 3D 모델 대신 이미지/애니메이션 기반 Walk-through 제공
- 반응형 웹 UI

## 실행 방법
1. 패키지 설치
   ```bash
   pip install -r requirements.txt
   ```
2. 앱 실행
   ```bash
   streamlit run Home.py
   ```

## 배포
- Streamlit Community Cloud에서 바로 배포 가능
